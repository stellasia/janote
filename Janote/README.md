# Janote

## Purpose

Provide a simple graphical interface for teachers to handle small group of students and their grades. 

## Developer guide

Application developed using java/swing and the sqlite databases. Packages include: 

- model
    - entities
    - db
        - connection
        - dao
        - managers
- view 
    - gui
